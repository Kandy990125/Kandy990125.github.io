配置环境

安装环境 Windows 10

  1. 下载并安装 Node.js & Git

    Node.js 下载地址：Node.js下载地址

    Git 下载地址：Git下载地址

    下载完成之后，打开下载目录，点开这两个文件，一路next 下去

简单介绍一下这两个工具：

npm 是 JavaScript 世界的包管理工具,并且是 Node.js 平台的默认包管理工具。通过 npm 可以安装、共享、分发代码,管理项目依赖关系。

Git 是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

  1. 安装 Hexo

    在桌面打开 Git Bash 快捷方式

    输入 npm install -g hexo-cli
    关闭 Git Bash

  2. 初始化 Hexo

    同时在 Hexo 相关存储的位置新建一个文件夹（没有特别要求，比如我在了D:/目录下新建了一个文件夹，命名为：Hexo），打开文件夹目录在空白处右键选择 Git Bash Here

    输入 hexo init
    输入 npm install

  1. 启动服务器访问Hexo网站

    输入 hexo s
    在浏览器中输入 localhost:4000
    成功打开 Hexo 网页
    下图来自：结合hexo在GitHub上搭建个人博客全过程

GithubPages

将网站挂载在Github服务器上，通过<Github账号>.github.io访问
  1. 创建Github账号 （已有账号此步跳过）

  2. 登录并点击右上角的 *+* 号，选择 new repository
    repository name 是 < Your Github name > .github.io
    比如我注册时的Github Name是Kandy990125，那么我的repository name就是 Kandy990125.github.io
    其他的选项不用勾选，下图显示红色是因为我已经有了这个仓库，第一次创建这个仓库的话不会有红色的提示。

  3. 将本地的Hexo文件夹推送到GithubPage

  • 安装 hexo-deployer-git 插件

    在 *Hexo* 文件夹下打开的 *Git Bash* 中输入：***npm install hexo-deployer-git --save***
此过程出现的*warning*可忽略，只要不出现*ERR*就是安装成功

  • 创建并添加 SSH Key

    • 在 Git Bash 中输入 ssh-keygen -t rsa -C “注册时使用的邮箱地址”
    • 用记事本打开打开本地文件夹 C:\Users\AdministratorName.ssh\id_rsa.pub 文件
      复制文件内容
      • 打开 New SSH Key 网页，点击右上角 New SSH Key，Title 可随意填写，将刚刚复制的文件内容粘贴在 SSH Key下
      • 测试是否添加成功。在命令行（即Git Bash）依次输入如下命令 ，返回“You’ve successfully authenticated”即成功
        ssh -T git@github.com
  • 用记事本打开 *D:/Hexo/_config.yml* 文件
    文件末尾修改为

    deploy:
    type: git
    repo: git@github.com:<Github账号名称>/<Github账号名称.github.io.git
    branch: master

    • 在命令行（即Git Bash）依次输入 【 hexo g、hexo d】， 返回 【 INFO Deploy done: git 】 即成功推送

    添加域名

  1. 在阿里云域名平台中购买域名
  2. 打开域名控制台查看自己拥有的域名
  3. 对新购买的域名进行实名认证
  4. 点击对应域名操作中的解析
  5. 点击添加记录

    类型选择为 CNAME；
    主机记录即域名前缀，填写为www；
    记录值填写为<Github账号名称>.github.io；
    其他不用更改

  6. 打开博客仓库设置 > Options >下滑找到 Github Page > Custom domain >填写刚刚购买的域名
  7. 打开 D:/Hexo/source 文件夹，创建并打开 CNAME （注意没有后缀！！），写入你的域名（如 www.evanlin.cn ）并保存。

Next 主题下的 Hexo 网站

Next 主题应用

  1. 下载Next主题
  2. 解压文件夹，将文件夹复制到 D:/Hexo/themes/ 目录下，重命名为 next
  3. 修改 D:/Hexo/_config.yml:

    *#Extensions

    ##Plugins: https://hexo.io/plugins/
    ##Themes: https://hexo.io/themes/
    theme: *next2**
  4. 执行 hexo clean，hexo d -g， hexo s，在浏览器中输入 localhost:4000 预览是否发生变化，没有发生变化，则删除 D:/Hexo/public 文件夹，再重复输入一遍
  5. 在浏览器中输入自己的域名，查看网页是否发生变化，如果没有发生变化，按 F12，F5，选择 Disable Cache

Next 主题美化

注意：每次修改后，查看自己的网页有没有改变，没有改变则执行 hexo clean，hexo d -g

1. 修改网站标题&作者信息

打开 D:/Hexo/_config.yml

#Site
title: Kandy990125’s blog #网站标题
subtitle:

#网站副标题
description: Your description

#网站描述
author: Kandy990125

#博主的名字
language: zh-Hans

#网站使用的语言
timezone: #网站时区 ，Hexo 默认使用您电脑的时区

2. 修改网站的前面的图标

  • 将要修改的图标文件放在 D:/Hexo/themes/next/source/images 文件夹下，如我的图标文件路径是：D:/Hexo/themes/next/source/images/jituibao.ico

  • 用记事本打开并修改 D:/Hexo/themes/next/_config.yml 为：

    favicon:
    small: /images/jituibao.ico
    medium: /images/jituibao.ico
    apple_touch_icon: /images/jituibao.ico
    safari_pinned_tab: /images/jituibao.ico

    3. 将主页头像设成可旋转的圆形

    • 首先设置显示主页头像

      • 将要修改的图标文件放在 D:/Hexo/themes/next/source/images 文件夹下
        如我的图标文件路径是：D:/Hexo/themes/next/source/images/avatar.jpg

      • 用记事本打开并修改 D:/Hexo/themes/next/_config.yml 为：

        #Custom Logo

        #Do not support Scheme Mist currently.
        custom_logo:
        enable: true
        image: /images/avatar.jpg

        • 用记事本打开 D:\Hexo\themes\next2\source\css_schemes\Muse\logo.styl
        • 修改文件为：

          .custom-logo {
          .site-meta-headline {
          text-align: center;
          }
          .brand {
          background: none;
          }
          .site-title {
          margin: 10px auto 0;
          font-size: 24px;
          color: $black-deep;
          a {
          border: none;
          }
          }
          .custom-logo-image {
          margin: 0 auto;
          padding: 5px;
          max-width: 150px;
          background: white;
          }
          .custom-logo-image
          {
          margin: 0 auto;
          padding: 5px;
          max-width: 150px;
          background: white;
          / start/
          border-radius: 50%;
          transition: 2.5s all;
          }
          .custom-logo-image:hover
          {
          transform: rotate(360deg);
          }
          / end /
