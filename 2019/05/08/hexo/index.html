添加域名

  1. 在阿里云域名平台中购买域名
  2. 打开域名控制台查看自己拥有的域名
  3. 对新购买的域名进行实名认证
  4. 点击对应域名操作中的解析

  5. 点击添加记录

  6. 打开博客仓库设置 > Options >下滑找到 Github Page > Custom domain >填写刚刚购买的域名

  7. 打开 D:/Hexo/source 文件夹，创建并打开CNAME（注意没有后缀！！），写入你的域名（如 www.evanlin.cn ）并保存。
